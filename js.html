Массивы

Массивы лучше объявлять через const

const array = [ 1, 2, 3, 4, 5 ]

array.length - длина масссива

array[ 1 ] - обращение к индексу

array[ array.length - 1 ] - последний элемент массива

array.splice( )

array.map() - перебирает все элементы массива. Всегда нужно использовать return

Возвращает новый массив с измененными данными.

                array.map( function ( item, index ) {
                    return item * 2
                } )

Работает быстрее forEach, т.к. возвращает новый массив, а не мутирует исходный.

array.forEach() -

Его лучше использовать, если нужно выводить данные, без изменения массива
